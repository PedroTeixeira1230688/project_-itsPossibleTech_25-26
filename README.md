# Projeto

## Como correr o projeto

### Backend
1. Entra na pasta: `cd project_-itsPossibleTech_25-26/backend`
2. Instalar as dependências: `npm install`
3. Inicia o servidor: `node index.js`

### Frontend
1. Entra na pasta: `cd project_-itsPossibleTech_25-26/frontend`
2. Instalar as dependências: `npm install`
3. Inicia a aplicação: `npm start`
